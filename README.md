Dari tugas yang telah saya kerjakan, saya dapat menyimpulkan bahwa Python sangat membantu dalam mengolah berbagai jenis data dengan cara yang mudah dipahami.

Pada bagian awal, saya menggunakan **list**, yaitu kumpulan data yang bisa diubah. Dalam tugas ini, saya menggunakan list untuk menyimpan beberapa data campuran, lalu melakukan berbagai manipulasi seperti menambah (append, insert, extend), menghapus (pop, remove), serta mengambil sebagian data menggunakan slicing. Hal ini menunjukkan bahwa list sangat fleksibel untuk mengelola data.

Selanjutnya, saya menggunakan **tuple**, yaitu data yang tidak bisa diubah. Dalam tugas ini, saya menggunakan tuple untuk menyimpan beberapa nilai tetap, lalu mengambil data berdasarkan indeks dan melakukan **unpacking** (memecah isi tuple ke beberapa variabel), sehingga lebih mudah digunakan.

Pada bagian **set**, saya melihat bahwa set hanya menyimpan data unik. Saat saya membuat set dengan data yang sama, duplikat otomatis hilang. Saya juga melakukan operasi seperti union, intersection, difference, dan symmetric difference untuk melihat hubungan antar data.

Kemudian, pada **dictionary**, saya menyimpan data mahasiswa dalam bentuk key dan value, seperti nama, nim, dan kota. Dalam tugas ini, saya juga menambahkan, mengubah, dan menghapus data, serta menampilkan seluruh isi dictionary menggunakan perulangan.

Di bagian **nested structure**, saya membuat list yang berisi beberapa dictionary (contohnya data buku). Dari situ, saya dapat menampilkan semua judul buku dan melakukan filter berdasarkan tahun menggunakan **list comprehension**, yaitu cara singkat untuk membuat atau memilih data dari sebuah list.

Pada bagian **comprehension**, saya membuat list angka genap dan kuadrat, dictionary untuk menentukan genap/ganjil, serta set untuk mengambil huruf unik dari sebuah kalimat. Hal ini menunjukkan bahwa comprehension membuat kode lebih singkat dan efisien.

Selanjutnya, pada bagian **function**, saya membuat fungsi seperti `greet`, `tambah`, dan `rata_rata` untuk menjalankan tugas tertentu. Function membantu agar kode lebih rapi dan bisa digunakan berulang kali.

Kemudian pada bagian **class**, saya membuat class `Student` yang berisi data dan fungsi terkait mahasiswa, seperti menambah nilai, menghitung rata-rata, dan menentukan status lulus. Hal ini menunjukkan konsep **OOP (Object-Oriented Programming)**, di mana data dan fungsi digabung dalam satu objek.

Pada bagian berikutnya, saya menggunakan **library**, yaitu kumpulan kode siap pakai. Saya menggunakan **NumPy** untuk mengolah data angka, seperti menghitung rata-rata, median, dan standar deviasi. Saya juga menggunakan **pandas** untuk membuat tabel data (DataFrame) yang berisi nama, nim, dan nilai mahasiswa, serta menentukan status lulus atau tidak.

Terakhir, saya menyimpan hasil ke file teks, yang menunjukkan bahwa Python juga dapat digunakan untuk menyimpan dan mengelola data secara nyata.
